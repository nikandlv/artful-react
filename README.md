### artful-react 

Artful is a company page design

Mad props to <a href="https://www.behance.net/elcan12">Elcan Huseynaliyev</a> for the design

The whole project was written in half of a day

#### Run

```
npm install

npm run dev
```

Visit `localhost:3000`

### Export

```
npm run build

npm run export
```
